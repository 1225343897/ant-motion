# Ant Motion of React

- order: 0
- index: true

---

这是 Ant Motion 的所有 React 组件的演示站点, 可结合 Ant Design 来实现更炫酷的页面结构。

## 特性

- 简单方便又能快速的搭建出页面里的动画效果。
- React Component 上的动效组件。
- 基于 npm + webpack + babel 的工作流，支持 ES2015。

## 如何使用

每个组件单个安装, 如:

```js
$ npm install rc-tween-one --save
```
